新闻文档会支持常见 Markdown 语法，并优先覆盖公告和技术笔记常用的排版。

<!-- more -->

## 支持的语法

| 类型 | 状态 |
| --- | --- |
| 标题 | 已规划 |
| 表格 | 已规划 |
| 任务列表 | 已规划 |

## 任务列表

- [x] 解析本地 Markdown
- [ ] 渲染文章详情
- [ ] 增加标题目录

## 链接

外部链接会在新窗口打开，避免打断当前阅读流程。
