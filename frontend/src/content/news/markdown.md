## 标题
**在markdown总共有六级标题。**
> # 一级标题
> ## 二级标题
> ### 三级标题
> #### 四级标题
> ##### 五级标题
> ###### 六级标题
代码部分
```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## 段落部分

### 换行
**markdown有两种换行的方式，在此使用【】来指代空格**  

> 【】【】\n
> 
>  \n \n  

第一种更倾向于段落向换行，可以在表内或者是引用内使用  
第二种更倾向于分隔换行，类似于另起一段，间距会比第一种的大

> * 第一种  
>   这是延续
> * 第二种
> 
> 他们不会相连

## 列表

### 有序列表
**在markdown中有序列表使用数字加点（`1. text`）来实现，例如**

> 1. 这是一个有序列表
> 2. 这是有序列表的第二项  
>    还可以进行表内换行
> 3. 结束 

代码部分：
```markdown
1. 这是一个有序列表
2. 这是有序列表的第二项  
   还可以进行表内换行
3. 结束 
   注意点后应该有空格
```

### 无序列表
**markdown中的无序列表使用`*`来实现，例如**

> * 这是一个无序列表
> * 这是第二项  
>   同样可以换行
> * 结束
代码部分
```markdown
* 这是一个无序列表
* 这是第二项  
  同样可以换行
* 结束
```

### 列表嵌套
**在列表中嵌套列表，只需要使用四个空格或者是tap就行**

> * 外无序列表
>     * 中无序列表
>       * 内无序列表
>         * 没有了

代码部分：
```markdown
* 外无序列表
    * 中无序列表
      * 内无序列表
        * 没有了
```

+ [ ] 这是一个勾选框

### 代码块以及引用
**一共有两种代码块，分别是行内代码块和block代码块**

要使用行内代码块的话使用` 1`
```markdown
` 1`
```

如果是多行的代码块就使用三个`+代码属于哪个
```python
  a = 1
  print("Hello World!")
```
```markdown
  ```python
    a = 1
    print("Hello World!")
  ```
```

> 如果想要使用引用的话，使用 > 
```markdown
> 要引用的内容
```

## 超链接
实现超链接有两种方式
* 第一种方式
  使用 [text]+(url)
  **例如**：点击跳转到我的[GitHub](https://github.com/Yiyan-Jiang)
  ```markdown
  点击跳转到我的[GitHub](https://github.com/Yiyan-Jiang)
  ```
* 第二种方式
  使用 [text][label] 引用，用于解决链接过长影响编辑
  [我的GitHub][a]
  ```markdown
  [我的GitHub][a]
  [a]: https://github.com/Yiyan-Jiang
    //注意这里[a]:应该紧贴，而且是写在所有代码块的后边，不写在里面
  ```
[a]: https://github.com/Yiyan-Jiang

## 标注
**实现标注的话，使用text【 ^1 】,然后【 ^1 】：**
例如
  如何访问我的 GitHub[^1]
[^1]: 我的 GitHub 地址是：https://github.com/Yiyan-Jiang

代码：
```markdown
    如何访问我的 GitHub[^1]
[^1]: 我的 GitHub 地址是：https://github.com/Yiyan-Jiang
```

## 表格

**使用表格的话，大概的模板如下**
```markdonw
|表头| 第二个表头 |
| -- | --------- |
|内容| 后续的内容 |
```

|表头| 第二个表头 |
| -- | --------- |
|内容| 后续的内容 |

| 左对齐 | 居中对齐 | 右对齐 |
| :----- | :------: | -----: |
| 内容1  |  内容2   |   内容3 |
| 左     |   中     |      右 |
```markdown
