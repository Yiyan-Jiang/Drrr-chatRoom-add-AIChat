## Numpy

NumPy (Numerical Python) 是一个开源的 Python 库，在科学和工程领域被广泛使用。NumPy 库包含多维数组数据结构，例如同质的 N 维 ndarray

使用下面这个命令去安装numpy包
```bash
uv pip install numpy
```
然后使用下面这行代码导入
```python
import numpy as np
```

### 如何创建numpy数组？

目前创建numpy数组主要有以下几种方式：
1. 使用py原生数组的导入
2. 预定义形状填值
3. 划分好距离取间隔
4. 对已有的numpy数组的变形
5. 线代中常规矩阵的创建
6. 随机矩阵的创建

```python
# 1. 使用原生的py数组导入
row_arr = [[1,2,3,4,5],[1,3,4,6,7]]
arr = np.array(row_arr)
```

```python
# 2. 预定好了形状来填值
arr = np.zeros((2,5),dtype=np.int64) # 创建全 0 矩阵
arr = np.ones((2,5),dtype=np.int64) # 创建全一矩阵
arr = np.empty((2,5),dtype=np.float64) # 创建空矩阵，空矩阵的初始化效率会更高
arr = np.full((2,5),value) # 也可以使用full去指定填充的数字
arr = np.zeros_like(Numpyarr) # 也可以使用like类去创建和Numpy数组结构一样的矩阵
arr = np.full_like(Numpyarr,value) # 同理，创建和Numpyarr结构一样的全为value的矩阵
```

```python
# 3. 划分好距离

# 如果已经知道数据范围，类似py原生的range，可以这样创建
arr = np.arange(1,10,2)
# 这里也是和range一样的左闭右开，第三个参数代表步长

# 如果不知道步长，只是想划分区间的份数，使用下面的方法
arr = np.linspace(1,10,5)

# 区间如果是对数的话，也可以使用下面的方法，其中base这个参数代表的是以什么数字为底
arr = np.logspace(1,10,5,base=2)
```
```python
# 4.对已有的numpy数组的变形

# 可以水平或者垂直去合并已有的数组
row_arr1 = np.array([1,2,3])
row_arr2 = np.array([7,8,9])

arr = np.hstack((row_arr1,row_arr2)) # 水平合并
arr = np.vstack((row_arr1,row_arr2)) # 竖直合并

# 也可以对数组进行切分
arr = np.array([[1,2,3,4],[1,3,5,6]])
newArr1, newArr2 = np.hsplit(arr,2)
# 这样会按照列切分成两个数组

# 也可以复制现有的数组
row_arr = np.array([1,2,3])
arr = np.copy(row_arr)
arr = np.view(row_arr) 
# 这里可以区分一下copy和view，copy类似深拷贝，改值是不会动原矩阵的，view类似浅拷贝，只是视图
```

```python
# 5. 线代中常规矩阵的创建

# 创建单位矩阵
arr = np.eye(3,3) # 创建一个三行三列的单位矩阵

arr = np.diag([1,2,3]) # 创建对角矩阵，不过要给出创建出的对角线值
```

```python
# 6.随机矩阵的创建

# 生成0到1 之间的随机浮点数，均匀分布
arr = np.random.rand(3,3)

# 也可以指定范围去生成随机矩阵
arr_float = np.random.uniform(5,10,size=(3,3))
arr_int = np.random.randint(1,10,size=(3,3))

# 生成正太分布的随机矩阵
arr = np.random.randn(3,3)

# 有的时候我们可能需要在调试的时候生成一样随机矩阵，这个时候可以设置随机种子
np.random.seed(3)
arr = np.random.randint(0,10,size=(3,3))

```

### Numpy 矩阵所具有的属性

Numpy矩阵主要的属性包括维度之类的，具体的api见下面

```python
arr = np.array([[1,2,3],[1,2,3]])

print(arr.ndim) # 矩阵的维度
print(arr.shape) # 矩阵的形状，即行和列
# 这里要注意一个点就是一维矩阵的shape指的是元素的个数，不是行和列
print(arr.dtype) # 存储元素的数据类型
print(arr.size) # 存储元素的个数
print(arr.itemsize) # 单个元素占用的内存字节数
print(arr.nbytes) # 数组总内存占用量 
print(arr.flags) # 内存存储的方式 
```


### Numpy矩阵的运算

和py的原生矩阵不太一样，Numpy的矩阵更像是我们常在线性代数里面做的运算，加减乘除都是两个矩阵的各个元素去对应的位置做加减乘除，如果要算矩阵乘法的话使用的是@这个运算符号

```python
raw_arr1 = np.array([[1,2,3],[1,1,1]])
raw_arr2 = np.array([[1,2,1],[1,3,1]])
# 计算矩阵乘法
arr = raw_arr1 @ raw_arr2
```
那么如果两个数组不一样大小，去做加减乘除运算会发生什么呢？（矩阵乘法不做讨论，线性代数的基本知识）

这里会引出Numpy的**广播**机制

当两个数组形状不同，但维度“兼容”时，NumPy 会把较小的数组“虚拟扩展”成较大的形状，然后逐元素做加减乘除。

从两个数组的形状最右边开始对齐，逐维比较：
1. 两个维度相等，可以广播
2. 其中一个维度是 1，可以广播
3. 否则报错

就是一维的部分会扩展（复制）一份相同的值去补到不同维度，使得双方矩阵的维度相同

### Numpy的切片和索引

Numpy 的切片和索引没什么好讲的，基本都和py的原生数组索引相同，唯一多的一个功能是numpy支持布尔索引，可以在`arr[arr > 3]` 这种方式取出数组中所有大于3的元素，同时布尔索引也是支持`|` `&` `~` 这类运算符的

### Numpy 中常用的函数和方法

基本的方法有挺多都是可以传入`axis`这个参数去特定要求某个维度的，比如arr.sum(axis=1)就是对第一个轴，也就是按行进行求和。这里的维度看的是`shape`，axis等于几实际就是对shape里面的第几个元素去做操作，注意依旧计算机索引从0开始

```python
# 基本的方法
arr.sum() # 求和
np.sqrt(arr) # 平方根
np.exp(arr) # 计算每个数的指数函数 e^x
np.log(arr) # 计算每个数的自然对数
np.isnan(arr) # 计算每个数是否为 NaN，返回布尔数组
np.mean(arr) # 平均值
np.median(arr) # 中位数
np.var(arr) # 方差
np.std(arr) # 标准差
np.max(arr) # 最大值
np.min(arr) # 最小值
np.percentile(arr,75) # 分位数
arr.astype(np.int64) # 转换数据类型
```

```python
# 比较函数
np.greater(arr,value) # 比较是否大于某个值，返回布尔矩阵
np.less(arr,value) # 比较是否小于某个值
np.equal(arr,value) # 比较是否等于某个值
np.where(arr<value,if1,if2) # 对矩阵进行分支判断处理，类似if-else
np.select([arr < value, arr > value],['小于','大于'],default='等于') 
# 类似多分支处理，default不是必须传，不传时默认值是0
```

```python
# 排序函数，和原生的一样是原地排序
arr.sort() 

# 拼接函数
np.concatenate([arr1,arr2])

# 分隔,这里注意一下如果传入的是整数，必须能被整除
np.split(arr,value)

# 重塑数组，把数组重塑成另外的size，但是注意这里元素的个数必须符合，不然会报错
np.reshape(arr,(3,1))

# 降维数组,其中 ravel() 往往更接近“视图”，flatten() 更偏向“拷贝”。
np.ravel(arr)
arr.flatten()


# 为数组添加新的维度
arr[np.newaxis,:] #newaxis 每次可以扩张一个新的维度
np.expand_dims(arr,axis=1) # expand_dims 可以在指定的位置添加维度，看的shape，注意一维数组的shape不是几行几列而是有几个元素
```

```python
# 反转数组
np.flip(arr)

# 转置数组
arr.T
```

import pandas as pd

### Pandas的基本数据结构

pandas 提供了两种数据处理类：

  1. Series：一种一维标记数组，可容纳任何类型的数据
  如整数、字符串、Python 对象等。

  2. DataFrame：一种二维数据结构，容纳数据，类似于二维数组或带有行和列的表。

下面会分成两类来讲：

### Series 类的创建以及常用的一些方法

其实学过Numpy之后会发现大多数方法都是通用的，Pandas最主要的处理csv这类文件会好一点

```python
# 创建Series一维数组，可以自定义索引的数组的名字
s = pd.Series([1,2,3,4,5,np.nan,None],
index=['a','b','c','d','e','f','g'],
name = 'The first try')

# 或者从原来的Series数组里面截取部分数据来创建
newSeriesArr = s.loc[['a','f']]
```

```python
# 下面是一些常用的方法
print(s.index)
print(s.values)
print(s.shape)
print(s.ndim)
print(s.size) 
print(s.loc['a']) # 显式索引,支持索引 at也可以
print(s.iloc[0]) # 隐式索引 ， iat也可以
print(s.mean())
print(s.std())
print(s.median())
print(s.var())
print(s.sum())
print(s.quantile(0.25)) # 百分位数字
print(s.mode()) # 众数

# 默认打印前后五行,可以传入参数来改变打印的行数
print(s.head())
print(s.tail())

# 查看所有的描述信息
print(s.describe())

# 获取非缺失值个数
print(s.count())

# 判断某个元素是否缺失
print(s.isna())

# 判断某个元素是否在集合里
print(s.isin([2]))
```

可以看到大部分函数其实都是相同的，可以大大减轻记忆的负担

```python
# 排序
print(s.sort_values()) # 按值排序
print(s.sort_index()) # 按索引排序

# 去重
print(s.drop_duplicates())
print(s.unique()) # 得到的是数组
print(s.nunique()) # 拿到了去重之后元素的总个数
```

### DataFrame 类的创建以及一些方法

```python
# 创建，传入一个对象，自定义索引名称
df = pd.DataFrame({'第一列': [1,2,3], '第二列': [4,5,6]}, index=[1,2,3])

# Series 有的很多查看和统计方法，DataFrame 也有类似用法

# 获取行索引
print(df.index)

# 获取列索引
print(df.columns)

print(df.head()) # 查看前 5 行
print(df.tail()) # 查看后 5 行
print(df.shape) # 查看行数和列数
print(df.columns) # 查看所有列名
print(df.info()) # 查看每列的数据类型和缺失情况
print(df.describe()) # 查看数值列的统计信息

# 取一列，结果是 Series
name_col = df['姓名']
 
# 取多列，结果是 DataFrame
small_df = df[['姓名', '年龄', '城市']]
 
# 按标签取行和列
value = df.loc[0, '姓名']
part = df.loc[0:3, ['姓名', '年龄']]
 
# 按位置取行和列
value = df.iloc[0, 1]
part = df.iloc[0:3, 0:2]

# 查看每一列缺失值数量
print(df.isna().sum())
 
# 删除包含缺失值的行
clean_df = df.dropna()
 
# 用指定值填充缺失值
df['年龄'] = df['年龄'].fillna(0)
 
# 用平均值填充缺失值
df['分数'] = df['分数'].fillna(df['分数'].mean())

# 按分数从高到低排序
sorted_df = df.sort_values('分数', ascending=False)
 
# 按多列排序
sorted_df = df.sort_values(['城市', '分数'], ascending=[True, False])
 
# 删除完全重复的行
unique_df = df.drop_duplicates()
 
# 按指定列去重
unique_user_df = df.drop_duplicates(subset=['用户ID'])
```

### 读取和写入文件

Pandas 最常用的入口就是从文件读取表格数据，尤其是 csv 和 Excel。

```python
# 读取 csv 文件
df = pd.read_csv('data.csv')
 
# 如果中文 csv 打开乱码，可以尝试指定编码
df = pd.read_csv('data.csv', encoding='utf-8')
df = pd.read_csv('data.csv', encoding='gbk')
 
# 读取 Excel 文件
df = pd.read_excel('data.xlsx')
 
# 保存结果
df.to_csv('result.csv', index=False)
df.to_excel('result.xlsx', index=False)
```

这里 `index=False` 的意思是不把 DataFrame 的行索引额外写入文件，一般导出清洗后的表格时都会加上。

当数据分散在多个表里时，经常需要把它们拼接或关联起来。

```python
# 上下拼接，要求列结构基本一致
all_df = pd.concat([df1, df2], ignore_index=True)
 
# 按 key 关联两个表，类似 SQL join
merged_df = pd.merge(user_df, order_df, on='用户ID', how='left')

# concat 更像是把多张结构相同的表堆起来，merge 更像是按照某个字段把两张表关联起来。
```

### Pandas 时间相关的函数

很多分析都离不开时间，比如按天、按月统计数据。

```python
# 把字符串转换成时间类型
df['创建时间'] = pd.to_datetime(df['创建时间'])
 
# 提取年份、月份、日期
df['年份'] = df['创建时间'].dt.year
df['月份'] = df['创建时间'].dt.month
df['日期'] = df['创建时间'].dt.date
 
# 按月份统计数量
monthly_count = df.groupby(df['创建时间'].dt.to_period('M')).size()
```

只要一列被转换成了 datetime 类型，就可以通过 .dt 取出年、月、日、星期等时间信息。
